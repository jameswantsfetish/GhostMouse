# Support

If you need help with GhostMouse, start with:

- [Setup Guide](docs/SETUP.md)
- [Windows Guide](docs/WINDOWS.md)
- [Troubleshooting](docs/TROUBLESHOOTING.md)
- [Architecture Notes](docs/ARCHITECTURE.md)

For bugs, open a GitHub issue and include:

- Your operating system
- Java version from `java -version`
- The exact command you ran
- Whether you used demo mode or the Python sidecar
- Any relevant logs or terminal output

For safety-related issues, see [SECURITY.md](SECURITY.md).
