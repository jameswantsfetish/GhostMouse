# Changelog

All notable changes to GhostMouse will be documented in this file.

## 0.1.0 - Initial Public Release

- Added Java 21 desktop app with Swing debug window and optional system tray.
- Added demo landmark provider for safe testing without a camera.
- Added socket landmark provider for MediaPipe sidecar integration.
- Added gesture support for movement, left click, right click, drag, and scroll.
- Added adaptive pointer smoothing and release-based click debouncing.
- Added Python MediaPipe sidecar for webcam landmark streaming.
- Added Maven build, unit tests, GitHub Actions CI, and project documentation.
